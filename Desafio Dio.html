A História do HTML

HTML é a sigla para Hypertext Markup Language, a linguagem padrão para criação de páginas na web. Sua história começa na década de 1990, quando Tim Berners-Lee, pesquisador do CERN, desenvolveu a primeira versão com o objetivo de compartilhar informações científicas de forma acessível.

A primeira versão do HTML surgiu em 1993, ainda em estágio embrionário. Recursos que hoje são comuns não eram suportados. Com o tempo, novas versões surgiram: HTML 2.0 (1995), HTML 4.01 (1999), e uma grande evolução veio com o HTML5 em 2014.
