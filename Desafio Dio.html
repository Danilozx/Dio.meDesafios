A Historia do HTML

Voltar

HTML foi a sigla para o Hypertext Markup Language, que viria a ser uma linguagem padrão para a criação de páginas na web. A história do HTML tem seu inicio na década de 90, quando o Tim Berners-Lee ainda pesquisador do CERN, criou a primeira versão do HTML com o objetivo de compartilhar informações científicas de forma fácil, dinâmica, e acessível para os demais usuários.

A primeira versão do HTML foi lançada no ano de 1993, mas a linguagem ainda se encontrava em estagio embrionario, logo muitos recursos comuns em dias atuais, não eram suportados. Com o passar dos anos foram elaboradas novas versões do HTML, sendo o HTML 2.0 em 1995 e o HTML 4.01 em 1999 Porém foi com o lançamento do HTML5 em 2014 onde houve uma evolução signicativa.

Oque são Tags?

Voltar

Assim como o nome sugere, o HTML é um hiper texto que utiliza marcações em sua estrutura, e essas marcações são popularmente conhecidas como Tags. As tags são elementos utilizados para estruturar e formatar o conteúdo de uma página web. As tags podem ser definidas por meio de símbolos angulares ("<" e ">") e são geralmente compostas por uma tag de abertura e uma tag de fechamento, que envolvem o conteúdo a ser formatado.

Diferença do HTML para as linguagens de programação

Voltar

O HTML é uma linguagem de marcação usada para estruturar e formatar o conteúdo de páginas web. HTML não pode ser considerado uma linguagem de programação, pelo fato de não oferecer a criação de lógica ou interatividade complexa na página.

A programação em si é o processo de criar algoritmos e código para executar tarefas específicas. As linguagens de programação no geral, permitem a criação de lógica complexa e interatividade na página, como validação de formulários, animações e manipulação de dados em tempo real.
